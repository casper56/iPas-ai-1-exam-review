# 🎓 iPAS AI 應用規劃師 (初級) 歷屆甄試複習系統

> **本專案為單一自包含 (Self-contained Single-file) HTML5 離線複習系統，收錄完整歷屆甄試考古題與學術樣題，並提供一鍵式中文解析與生成工具。**

本系統嚴格遵循專業的前端軟體規格（V3.1 ~ V3.7 技術規範），內建「專家知識庫離線詳解引擎」，旨在為考生提供最極致、最流暢且最科學的備考體驗。

---

## 🚀 專案核心特色 (Core Features)

1. **📚 290 題精選考古題庫 (智能模糊去重)**
   - 完整收錄 **114 年第四梯次正式試題**、**115 年第一次正式試題** 以及 **114 年 3 月/9 月歷屆官方樣題**。
   - 內建 Python 智能文本去重演算法，自動比對題目相似度，**排除 34 題重複樣題**，並自動合併多個年份與來源標記，拒絕無效刷題。

2. **💡 專家知識庫離線詳解引擎**
   - 內建 24 個核心 AI 知識考點（如 K-means 限制、過擬合正則化、ROC/AUC 曲線 FPR/TPR 權衡、GAN 博弈、SMOTE 特徵合成、No-code 影子 IT 安全風險、個資合規限制等）的**專家解析規則引擎**。
   - 作答後自動顯示極具學術感的專業中文詳細解析，**100% 離線運作**，無網路連線依然能完美複習。

3. **📐 完美 UI/UX 佈局 (V3.4 & V3.1 規格)**
   - **全屏外擴自適應**：題目卡片容器使用 `width: calc(100% - 60px); margin: 20px auto; max-width: none;`，移除 1200px 限制，確保左右精確留白 30px，缩放時保持完美對齊。
   - **两欄式選項排版**：選項以整齊的美觀兩欄式網格 (2-column grid layout) 呈現。
   - **去粉紅代碼區塊 (Anti-Pink V3.1)**：所有程式碼（如 C++、Python 等）統一採用深炭色背景 `#1e293b` 與亮灰色文字 `#e2e8f0`，拒絕任何粉紅配色。

4. **🔒 專業正確答案條 (V3.1 規格)**
   - 復刻 PDF 1:1 學術感，採用 **「純白背景、綠色文字 (#198754)、左側 5px 綠實線」**。
   - 答案字母預設與背景同色（視覺隱藏），滑鼠移入 (Hover) 或反白選取時自動顯現，防止劇透。

5. **🔄 DOM 相對定位作答與重作機制 (V3.5 規格)**
   - 排除多個考區在同一個 DOM 樹中產生的 ID 衝突，高亮點選與狀態鎖定均採用 DOM 相對定位。
   - 提供「🔄 重新作答」按鈕，點擊後一鍵清除高亮、隱藏正確答案與詳細解析並解鎖，方便重複練習。

6. **📊 Collapsible Glassmorphic 進度側欄 (V3.6 規格)**
   - 懸浮於左側的毛玻璃進度追蹤側欄，不推擠主佈局。
   - **「🟩🟥🟧⚪」四色狀態進度網格**：🟩 答對 / 🟥 答錯 / 🟧 曾錯但已改正（特色功能！）/ ⚪ 未答。
   - **焦點引導**：點選題號自動平滑捲動，且目標題卡會**閃爍藍色邊框 1.5 秒**以瞬間引導視線。

7. **⏱️ 模擬考區 (Mock Exams)**
   - 提供隨機組卷測驗，自動從庫中抽選 50 題（科目一、二隨機各 25 題）。
   - 提供 60 分鐘倒數計時與交卷計分。交卷後自動判定全卷對錯，並**一次顯現全卷的所有詳細解析**。

8. **💾 全面資料持久化 (State Persistence)**
   - `localStorage` 自動保存您的「標記易錯題 (`flagged_questions`)」、「詳細作答歷史歷程 (`practice_history`)」以及「明亮 / 炭極深 / 護眼沙 3 種主題設定」。

## ⚙️ 一鍵生成與建置說明 (One-Click Builder)

本專案將所有的資料庫解析、去重合併與網頁渲染模板高度封裝於 `build_exam_system.py` 中。

### 1. 安裝依賴 (可選)
若您需要重新從 PDF 檔案中擷取文字，您可以使用 PyPDF 等套件（專案已內建提取好的 `scratch/txt/` UTF-8 純文字檔，因此**預設不需安裝任何第三方套件即可執行生成**）：
```bash
pip install pypdf
```

### 2. 執行一鍵建置
在專案根目錄下，使用 Python 3 執行建置腳本：
```bash
python build_exam_system.py
```

## 🚀 CI/CD 自動部署 (GitHub Actions - V3.7 規格)

本專案完美整合了 GitHub Actions 工作流。當您將變更 `push` 推送到 GitHub 的 `main` 分支時，系統將自動開啟工作流，進行靜態網頁編譯與自動部署至 **GitHub Pages**。

### 1. 工作流設定檔案路徑
`.github/workflows/deploy.yml`

### 2. GitHub 專案設定要求
在您的 GitHub 倉庫中，請確保進行以下設定：
1. 前往 `Settings` -> `Pages`。
2. 將 `Build and deployment` -> `Source` 設定為 **`GitHub Actions`**。
3. 工作流執行成功後，您即可透過 `https://casper56.github.io/iPas-ai-1-exam-review/` 線上無痛刷題與複習！
